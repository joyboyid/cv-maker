# Berkontribusi

Terima kasih sudah tertarik membantu **CV Satu Halaman**! Proyek ini gratis dan open source — setiap kontribusi membuat tool ini lebih berguna bagi pelamar kerja di Indonesia.

## Cara mulai

1. Fork repository ini
2. Clone ke mesin lokal:
   ```bash
   git clone https://github.com/YOUR_USERNAME/cv-satu-halaman.git
   cd cv-satu-halaman
   npm install
   npm run dev
   ```
3. Buat branch untuk perubahanmu:
   ```bash
   git checkout -b fitur/nama-fitur
   ```
4. Commit dan push, lalu buka Pull Request

## Ide kontribusi

- Template CV baru
- Terjemahan / perbaikan copy Bahasa Indonesia & Inggris
- Perbaikan aksesibilitas dan mobile UX
- Optimasi export PDF
- Dokumentasi dan contoh CV per profesi

## Standar kode

- Jalankan `npm run lint` sebelum PR
- Pastikan `npm run build` sukses
- Fokus pada perubahan kecil dan jelas — satu PR per fitur/fix

## Pelaporan bug

Buka [GitHub Issues](https://github.com/YOUR_USERNAME/cv-satu-halaman/issues) dan sertakan:

- Langkah reproduksi
- Perilaku yang diharapkan vs yang terjadi
- Browser dan perangkat (jika relevan)
